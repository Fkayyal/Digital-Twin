Navigeren
  • Zoom in/uit: Scroll met het muiswiel.
  • Bewegen: Houd de linkermuisknop ingedrukt en sleep om het beeld te verplaatsen.
Polygonen
  • Polygoon tekenen: Linksklik om punten te plaatsen, rechtsklik om af te sluiten (wordt daarna opgeslagen).
  • Info tonen: SHIFT + linksklik op een polygoon.
  • Hoogte verhogen: CTRL + linksklik op een polygoon.
  • Hoogte verlagen: CTRL + rechtsklik op een polygoon.
  • Verwijderen: ALT + linksklik op een polygoon (niet Spoordok).
